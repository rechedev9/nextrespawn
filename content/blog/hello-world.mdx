# Welcome to Your Blog

This is a sample post to demonstrate the MDX blog. You can write content using standard Markdown syntax with JSX support.

## Adding New Posts

Create a new `.mdx` file in `content/blog/` with the following frontmatter:

```yaml
title: "Your Post Title"
description: "A short description for SEO."
date: "2025-01-15"
author: "Your Name"
image: "/optional-og-image.png"
```

Posts appear automatically at `/blog/[slug]` with OG tags, JSON-LD structured data, and a sitemap entry.

## Markdown Features

You can use **bold**, *italic*, `inline code`, and more:

- Unordered lists
- [Links](https://example.com)
- Images via standard Markdown syntax
