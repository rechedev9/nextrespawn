# Hello, World

This is the first blog post rendered by ShipFast using **next-mdx-remote** in RSC mode.

## What is ShipFast?

ShipFast is a personal SaaS boilerplate that eliminates the first 20–40 hours of every new web project:

- **Auth** — Google OAuth + Magic Link via Auth.js v5
- **Payments** — Stripe checkout, webhooks, and subscription management
- **Email** — React Email templates sent via Resend
- **Blog** — MDX-powered with automatic SEO and JSON-LD
- **Database** — Prisma 7 + Supabase Postgres

Clone it, configure `config.ts`, and start shipping features.
